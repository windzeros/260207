# 내 이야기를 들어줘!

## 프로젝트 설명

24시간 심리 상담을 해주는 웹페이지입니다.
언제 어디서든 브라우저만 있으면 따뜻한 심리 상담을 받을 수 있습니다.

## 사용 기술

- **HTML** - 웹페이지 구조
- **CSS** - 스타일 및 디자인
- **JavaScript** - 상담 기능 및 사용자 상호작용

## 파일 구조

```
260207/
├── CLAUDE.md          # 프로젝트 설명 문서
├── index.html         # 메인 웹페이지 (HTML 구조)
├── style.css          # 스타일시트 (디자인 및 레이아웃)
└── script.js          # 자바스크립트 (상담 로직 및 상호작용)
```

## 실행 방법

브라우저에서 `index.html` 파일을 열면 바로 사용할 수 있습니다.

## 코딩 규칙

- 모든 주석은 **한국어**로 작성합니다.
- 초보자가 이해하기 쉽게 코드를 작성합니다.
- 모든 코드에 설명 주석을 달아 코드의 역할을 명확히 합니다.
